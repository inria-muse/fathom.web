Fathom Tools

You can access several network troubleshooting and monitoring tools by clicking on the Fathom icon on the toolbar (or via the Firefox menu on Android).

  • Debug My Connection: test your network configuration and connectivity if you are having problems with your network.
  • Home Network Discovery: discover devices and network connectivity in your home network.
  • Monitoring Graphs: system and network performance over time (see next section for details on background monitoring).

Background Monitoring

Fathom collects two types of background monitoring data:

  1. Performance metrics about your system (CPU, load, memory) and network connectivity (signal quality on Wifi, cross-traffic and network delays). These metrics are recorded periodically.
  2. Page load performance when browsing the web (time to fetch the content and render the page).
The monitoring is light-weight and should not impact your network browsing.

You can visualize the collected performance data via the Monitoring Graphs tool to identify problems and to compare your current performance with the past and in different network environments (if your device connects to multiple networks). In addition, we are currently working on to use this monitoring data to improve the diagnosis of the Debug My Connection tool (e.g. we plan to pin-point large deviations from the baseline automatically to help to solve problems).

You can always disable background monitoring via the Fathom menu. Optionally, you can share the monitoring data with us for research purposes (see next section on our data collection policy).

Data Collection

The Fathom project is a research project and we hope to collect data on real home networks configuration, performance characteristics and typical problems as part of the work. The extension will ask you for your permission to upload the results of the various network measurements to our secure server when you first use each tool. You can change the settings at anytime via the addon preferences (go to Menu -> Addons -> Fathom).

We do not collect data from third party scripts using Fathom (as explained below in 'Fathom Web Pages') and the information in this section applies only to data collected by the built-in Fathom tools. However, Fathom scripts may collect and upload data from your browser -- review the security manifest carefully!

Fathom Web Pages

Fathom provides network performance monitoring and measurement functionality for any web page via javascript. Each page using Fathom must provide a description of the required functionality and network destinations it will communicate with. Fathom will prompt for your approval of the requested features before the page is allowed to access them.
