Fathom Tools

You can access several network troubleshooting and monitoring tools via the Fathom Menu in the toolbar (or via the Firefox menu on Android):

  • Pageload Performance displays the pageload time of the current active tab.
  • Debug My Connection test your network configuration and connectivity if you are having problems with your network.
  • Home Network Discovery discover devices and network connectivity in your home network.
  • Monitoring Graphs explore web browsing, network and system performance over time.

Pageload Performance Monitoring

Fathom monitors on the background the pageload times of the sites you visit (except when in private-browsing mode). You can check the pageload performance of the current active tab in the Fathom Menu and historical data using the Monitoring Graphs tool.

The monitoring is light-weight and does not impact your internet browsing. However, you can disable the monitoring at the Fathom preferences at anytime.

System Performance Monitoring

Fathom monitors on the background the performance of your system (CPU, load, memory) and network connectivity (signal quality on Wifi, cross-traffic and network delays). You can explore the performance statistics using the Monitoring Graphs tool.

The monitoring is light-weight and does not impact your internet browsing. However, you can disable the monitoring at the Fathom preferences at anytime.
