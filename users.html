Fathom Tools

You can access several network troubleshooting and monitoring tools via the Fathom Menu at the toolbar (or via the Firefox menu on Android):

  • Pageload Performance displays the pageload time of the current active tab.
  • Debug My Connection test your network configuration and connectivity if you are having problems with your network.
  • Home Network Discovery discover devices and network connectivity in your home network.
  • Monitoring Graphs explore browsing and network performance over time.

Pageload Performance Monitoring

Fathom monitors on the background the pageload times of the sites you visit (except in private-browsing mode). The monitoring is light-weight and does not impact your network browsing. However, you can disable the monitoring via the Fathom preferences at anytime.

You can check the pageload performance of current active tab in the Fathom Menu .

Optionally, you can contribute pageload monitoring data for our research. By default the names of the sites you visit are all anonymized. This means that we cannot know which sites you visit or if you and another Fathom user have visited the same sites. However, in order to be able to study the network performance across users and various networks, we do not anonymize the identify of a small set of popular domains whitelisted domains (such as google.com). You can visualize and edit the whitelist at the extension preferences at any time.

System Performance Monitoring

Fathom monitors on the background the performance of your system (CPU, load, memory) and network connectivity (signal quality on Wifi, cross-traffic and network delays). The monitoring is light-weight and does not impact your network browsing. However, you can disable the monitoring via the Fathom preferences at any time.

You can explore the performance statistics using the Monitoring Graphs tool.

Optionally, you can contribute pageload monitoring data for our research. The system performance monitoring data does not contain any personally identifiable information.

Data Collection

Fathom is a research project and we use the extension to collect data for our research. However, all data collection is optional and you can use the Fathom extension and tools without sharing any data.
