Fathom is a Firefox extension that allows you to monitor and troubleshoot network performance problems. Fathom features:

  • For Users: Fathom has a number of built-in tools for network monitoring and troubleshooting.
  • For Developers: Fathom provides a javascript API that you can use on your own web pages for network performance measurements.

Fathom is developed by MUSE team at Inria Paris-Rocquencourt and International Computer Science Institute (ICSI) at Berkeley (CA, US) as a part of our research that studies user experience and performance of home networks and internet applications.

Install Fathom

Note, Fathom currently only works in the Firefox browser that you can get from Mozilla!
