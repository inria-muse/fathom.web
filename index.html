Fathom is a Firefox extension that allows you to monitor and troubleshoot network performance problems.

Who is behind Fathom ?

  • MUSE team at Inria Paris-Rocquencourt
  • International Computer Science Institute (ICSI) at Berkeley (CA, US)

Why should you install Fathom ?

  • Users: Fathom has a number of built-in tools for network monitoring and troubleshooting.
  • Developers: Fathom provides a javascript API that you can use on your own web pages for network performance measurements.

Install Fathom
Before installing, make sure you are running the latest Firefox browser from Mozilla! All platforms except Firefox for Android (coming soon) are supported.
