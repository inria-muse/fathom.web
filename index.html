Introduction

The Fathom project explores the browser as a platform for network measurement and troubleshooting. Fathom is a joint project between MUSE team at Inria Paris-Rocquencourt and International Computer Science Institute (ICSI) at Berkeley (CA, US). This web-site is about a re-write of Fathom on top of the Mozilla Add-on SDK, you can access the original Fathom project site on the ICSI server.

Download Fathom

Fathom is a Firefox extension so it only works on Firefox browser. A current test version of Fathom is available here. Use at your own risk :)

Use Fathom

You can read more about the Fathom tools and features on the For Users tab.

If you are interested in using Fathom on your web page, check out the For Developers tab and the API Documentation.
