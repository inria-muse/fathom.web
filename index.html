Fathom v2.0

Browser-based Network Measurement Platform

Introduction

The Fathom project explores the browser as a platform for network measurement and troubleshooting. Fathom is a joint project between MUSE team at Inria Paris-Rocquencourt and International Computer Science Institute (ICSI) at Berkeley (CA, US).

Download and Install Fathom

Fathom is a Firefox extension so it only works on Firefox browsers. You can get the latest Firefox browser from Mozilla! A beta test version of Fathom is available here (tested on OS X and Linux).

Fathom Tools

You can access several network troubleshooting and monitoring tools by clicking on the Fathom icon on the toolbar (or via the Firefox menu on Android).

  • Debug My Connection: test your network configuration and connectivity if you are having problems with your network.
  • Home Network Discovery: discover devices and network connectivity in your home network.
  • Monitoring Graphs: system and network performance measurements over time (see next section for details on background monitoring).

Background Monitoring

Fathom collects periodically basic performance metrics about your system (CPU, load, memory) and network connectivity (signal quality on Wifi, cross-traffic and network delays). You can visualize the monitoring data via the Monitoring Graphs tool. In addition, we use this monitoring data to improve the diagnosis of the Debug My Connection tool.

You can always disable the background monitoring via the Fathom menu. Optionally, you can share the monitoring data with us for research purposes (see next section on our data collection policy).

Data Collection

The Fathom project is a research project and we hope to collect data on real home networks configuration, performance characteristics and typical problems as part of the work. The extension will ask you for your permission to upload the results of the various network measurements to our secure server when you first use each tool. You can change the settings at anytime via the addon preferences (go to Menu -> Addons -> Fathom).
