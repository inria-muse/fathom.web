Fathom is a Firefox extension that allows you to monitor and troubleshoot network performance problems. Fathom is developed by MUSE team at Inria Paris-Rocquencourt and International Computer Science Institute (ICSI) at Berkeley (CA, US) as a part of our research that studies user experience and performance of home networks and internet applications.

What can you do with Fathom ?

  • Users: Fathom has a number of built-in tools for network monitoring and troubleshooting.
  • Developers: Fathom provides a javascript API that you can use on your own web pages for network performance measurements.

Install Fathom
Before installing, make sure you are running the latest Firefox browser from Mozilla! All platforms except for Firefox for Android (coming soon) are supported.
