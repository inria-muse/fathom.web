What Data is Collected by Fathom ?

All data collected by Fathom contains the following common information about the extension itself, the browser and the current network environment to which the computer is connected:

  • Fathom version and random installation ID
  • Browser version, operating system, time-of-day and browser timezone
  • Default network interface configuration (name, IP, mac)
  • If wireless, SSID and BSSID
  • Default gateway (IP, mac)
  • Public IP address visible to our server
  • ISP, AS number and coarse geo-location based on the public IP
  • IP addresses of the first three IP hops

System Performance Monitoring

  • Network interface statistics (bytes and packets sent/received)
  • Wireless network performance (signal/noise levels, rx/tx rates)
  • CPU load
  • Memory use (available, used, free)
  • DNS lookup measurement (random hostname)
  • RTT (ping) measurement to localhost, network gateway (1st hop), 2nd and 3rd IP hops, and a measurement server
  • Traceroute to a measurement server

Pageload Performance Monitoring

  • Accessed page's hostname, IP (in clear if whitelisted else anonymized)
  • Protocol (http or https)
  • HTML Navigation Timing: reports main document network performance timings (DNS request, connection setup, time-to-first-byte, pageload, etc.)
  • HTML5 Performance Timeline: reports the network performance timings for objects included in the main page.

Debug my Connection

  • Network interface(s) configuration
  • Routing table
  • DNS configuration
  • DNS lookup measurement (www.google.com)
  • Reachability test (with ping) to the default gateway, www.google.com and a measurement server
  • HTTP GET request to www.google.com and a measurement server

Home Network Discovery

  • Network interface(s) configuration
  • Default gateway (routing table)
  • Devices recently active in the network (ARP table)
  • Neighboring UPnP devices (via UPnP discovery)
  • Neighboring mDNS devices (via mDNS discovery)
