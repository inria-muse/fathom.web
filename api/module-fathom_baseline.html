Methods

(static) get(range, what)

Get historical baseline measurements data. Returns approximately the last 'range' of values (if available). To fetch a single metric, you can use the helper methods.
Parameters:
Name Type Description
range string One of 'day' (default), 'week', 'month', 'year'.
what Array List of 'cpu', 'load', 'tasks', 'mem', 'traffic', 'wifi' and 'rtt'. Undefined or empty list defaults to all metrics (execpt 'env' which has to be requested separately always -- see getEnv).
